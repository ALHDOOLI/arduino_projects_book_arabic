# كتيب تجارب أردوينو
كتيب تدريبي لأردوينو يحتوي على العديد من المشاريع التطبيقية للمبتدئين.

يحتوي الكتيب على 72 تجربة لتعلم كيفية برمجة أردوينو لأغراض متنوعة.

جميع صور الدوائر والأكواد متاحة في المجلد المرفق.
